# jayden-framework-go
##
本项目流程及目的：业务数据保存mysql后，发送rabbitmq，启动go协程消费队列数据，将数据推送到es，业务多条件查询时，直接查es，解决大数据查询慢的问题。 
gin框架，mvc架构（controller-service-model）
   
 ```shell script
redis->mysql->rabbitmq(kafka)->go comsume->elasticsearch
```

项目架构：  
```yaml script
app:
    config: app.yaml 项目核心配置
    constants: 常量配置
    controllers 控制器
    events 事件
    libs 类库
    logs 运行日志
    models 模型
    routers 路由
    services 服务层
    utils 助手类库
    main.go 程序入口文件
```

版本：
```go
go version 1.19
gin v1.8.1
```
